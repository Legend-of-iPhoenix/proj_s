(basic) HTML Syntax Guide

HTML uses tags to define styling. All tags are wrapped with angle brackets (< and >).

This means that the less than and greater than signs need to be escaped to be shown literally. You can use the text escaper tool at the bottom of this page to escape text.

Surround Paragraphs with <p> and </p>

Add extra line BReaks (newlines) with <br>. (The text escaper tool can handle this)

Use <p> and </p> to wrap Italicized text

<b> and </b> make text Bold

For Underlines, use <u> and </u>

Links/Anchors are different. Use <a href="link">text</a>

Unordered Lists are wrapped with <ul> and </ul>, with each List Item wrapped with <li> and </li>

Ordered Lists are the same as unordered lists, but they use <ol> and </ol> tags.
