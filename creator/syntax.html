Syntax

a lesson in basic HTML


Surround Paragraphs with <p> and </p>

Use <p> and </p> to wrap Italicized text

<b> and </b> make text Bold

For Underlines, use <u> and </u>

Links/Anchors are different. Use <a href="link">text</a>

Unordered Lists are wrapped with <ul> and </ul>, with each List Item wrapped with <li> and </li>
